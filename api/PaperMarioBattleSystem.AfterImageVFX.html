The amount to modify the alpha of each after-image by.

If FadeOff, closer after-images are rendered more opaque than further ones. If Constant, all after-images have the same alpha value.

FramesBehind

How many frames behind the BattleEntity's position each successive after-image is. Lower values result in after-images closer to the BattleEntity.

For example, if 2, the closest after-image would be 2 frames behind the BattleEntity's current position. The next closest after-image would be 4 frames behind, and so on.
