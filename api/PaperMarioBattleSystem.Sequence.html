Class Sequence

A Sequence used in a MoveAction.

Sequences define a series of step-by-step actions that are performed for a particular move.

For example, for Mario's Hammer, the Sequence would play Mario's run animation, move Mario up to the enemy, start the Action Command, then damage the enemy based on the result.

OnMiss()

How the action handles a miss. The base implementation is to do nothing, but actions such as Jump may go to the Miss branch.

The return value indicates whether missing continues testing for hitting the remaining BattleEntities or not. true means to continue, and false means to stop. The base implementation returns true.
