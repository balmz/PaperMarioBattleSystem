Enum Enumerations.AdditionalProperty

Namespace: PaperMarioBattleSystem
Assembly: PaperMarioBattleSystem.dll
Syntax
public enum AdditionalProperty

Fields

Name Description
AttackFXSounds

Tells that this BattleEntity plays sounds from Attack FX badges when dealing damage.

This should have an AttackFXManager as the value. Initialize it when adding this property, and clean it up when removing it.
AutoActionCommands

Tells that this BattleEntity should automatically complete Action Commands, provided they're enabled.

This should have an integer as the value, as it may be modified in multiple locations.
AutoStylishMoves

Tells that this BattleEntity should automatically complete Stylish Moves.

This should have an integer as the value, as it may be modified in multiple locations.
ChargedDamage
ConfusionPercent
DefendedByEntity

Tells that there is a BattleEntity defending this one and taking the hits for it. The only example in the PM games is the Shell from Koops' Shell Shield.

This will take in the BattleEntity defending this one as the value. This is inactive during this BattleEntity's phase to allow itself and allies to positively affect it, and active otherwise. If active, the BattleEntity defending this one will be targeted by opponents instead.
DipItemTurns

The number of Item turns the BattleEntity has. Set when using an item in the Double and Triple Dip menus.
HelperEntity

Tells that this BattleEntity is a helper or part of another BattleEntity.

This should have a BattleEntity as the value. The BattleEntity will be the BattleEntity that this one supports.
Immobile

Whether a BattleEntity is immobile or not. The BattleEntity's defensive actions are disabled and it cannot move.
Invincible
LightSource

Tells that this BattleEntity is a light source. If a BattleEntity with this property is in a dark battle, a region around it will be lit.

The value should be a double with the radius of its light.
NeverShowHP

Tells that this BattleEntity should never have its HP shown despite the value of ShowHP.

This should have an integer as the value, as it may potentially be modified in multiple locations.
ShowHP

Used for enemies who are tattled or if Peekaboo is active. This tells them to show their HP underneath them.

Use an integer for the value so removing Peekaboo doesn't remove this property from enemies if they have been tattled.
ShowStylishTimings

Tells that Stylish Move timings should be shown for this BattleEntity.

This should have an integer as the value, as multiple Timing Tutor badges can be equipped.
Untargetable

Tells that this BattleEntity is, in general, not targetable. Certain moves may still be able to target BattleEntities with this property.

Some BattleEntities innately have this (Ex. Bobbery Bombs), but it can also be applied dynamically through darkness or other types of battle settings.
