Class ActionCommand

AutoComplete

Whether the Action Command should automatically be successfully performed or not.

Since each action command is different, this will need to be manually implemented in each one. This may not always work, as the absolute success of some commands may be arbitrary (Ex. Art Attack, Bomb Squad).
