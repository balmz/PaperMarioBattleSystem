Class Mystery

The Mystery item. It uses a random item.

This chooses the set of random items from PM: { Mushroom, Super Shroom, Fire Flower, Stone Cap, Dizzy Dial, Thunder Rage, Pebble (falls down on Mario) }

If an item in question cannot be used (Ex. dark battles), it is removed from the roulette!
