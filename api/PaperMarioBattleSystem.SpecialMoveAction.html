Class SpecialMoveAction

The base class for Special Moves, which use up Star Power.

Special Moves are unaffected by All Or Nothing; make sure AllOrNothingAffected is false when defining damage data.

Note that all Crystal Star Special Moves can hit entities with the Invisible Status Effect.
