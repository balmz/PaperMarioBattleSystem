Class TimedLightCommand

An Action Command for timed lights. As the bar fills up, press a button when the lights are lit. One move that uses this is Yoshi's Mini-Egg.

The bar fills over time. As such, the values passed in should be time-based. Lights are spaced out across the bar according to the LightDistribution used.
